# v0.2.0 版本说明

发布日期：`2026-03-24`

## 这次版本做了什么

`v0.2.0` 把这个技能从“只谈结构”升级成“结构 + 流程”并重。

## 新增内容

1. 新增 [references/经典治理流程库.md](../references/经典治理流程库.md)
2. 新增 [references/反方质询与收敛回路.md](../references/反方质询与收敛回路.md)
3. 新增流程库相关包装说明和发布文案

## 主要变化

1. `SKILL.md` 开始显式推荐默认流程和条件流程
2. 输出约定增加“推荐经典流程库”
3. 场景手册和共享模式说明同步补齐

## 这一版沉淀下来的关键流程

- `discussion -> decision_package -> task_or_decision`
- `mechanism_review`
- `adversarial_inquiry -> defense -> convergence`
- `proposal -> validation -> shadow/canary -> activation_or_rollback`

## 向后兼容性

原有结构判断仍然保留，只是在此基础上补上流程层。

## 推荐发布标题

`v0.2.0 - 经典治理流程库`
