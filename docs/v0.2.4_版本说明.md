# v0.2.4 版本说明

发布日期：`2026-03-24`

## 本次补丁做了什么

`v0.2.4` 把文档膨胀从隐含问题提升成显式治理问题。

## 主要变化

1. 新增 [references/文档生命周期与压缩.md](../references/文档生命周期与压缩.md)
2. 在流程库中加入 `growth_signal -> lifecycle_review -> compact_or_archive`
3. 在输出约定中加入文档生命周期与压缩策略区块
4. 新增一份明确的文档膨胀质询记录

## 为什么这次补丁重要

因为文档一旦开始膨胀，恢复链和当前状态判断成本会很快抬高。
如果没有显式策略，团队通常会一直拖到已经很难接手才动手。

## 推荐发布标题

`v0.2.4 - 文档生命周期与压缩`
