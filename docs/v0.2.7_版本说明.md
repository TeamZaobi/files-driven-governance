# v0.2.7 版本说明

发布日期：`2026-03-25`

## 本次补丁做了什么

`v0.2.7` 把“继续开发”“开始审计”“反思”“推进”这类短口令，正式提升成可迁移的上游约定。

## 主要变化

1. 新增 [references/意图触发约定.md](../references/意图触发约定.md)
2. 区分主意图、别名层和修饰项层
3. 在输出约定中加入意图触发与执行约定区块
4. 说明书、元数据和包装文案同步更新

## 为什么这次补丁重要

因为短口令如果只活在聊天习惯里，就会随着工具、操作者和上下文变化而偷偷变义。
把它提升成上游约定，才能真正跨工具复用。

## 推荐发布标题

`v0.2.7 - 意图触发约定`
