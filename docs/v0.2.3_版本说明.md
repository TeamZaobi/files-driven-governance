# v0.2.3 版本说明

发布日期：`2026-03-24`

## 本次补丁做了什么

`v0.2.3` 把默认输出改成“核心必答 + 条件展开”，避免技能一上来就吐完整治理大图。

## 主要变化

1. 输出约定拆成核心区块和条件区块
2. `SKILL.md` 明确要求只展开真正相关的治理模块
3. 说明书和包装文档同步改成精确输出定位

## 为什么这次补丁重要

因为不是每个项目都需要整套重治理方案。
输出越精确，认知负担越低，建议也越容易落地。

## 适用场景

- 轻量项目
- 早期项目
- 只需要解决局部治理问题的项目

## 推荐发布标题

`v0.2.3 - 精确输出`
