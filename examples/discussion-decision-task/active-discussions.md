# Active Discussions

This is the visible entry for a discussion that is still active but already has a controlled promotion path.

## How to read this pack

1. [discussion.md](discussion.md)
2. [issue_ledger.md](issue_ledger.md)
3. [decision_package.md](decision_package.md)
4. [task_or_decision.md](task_or_decision.md)

## Current example topic

- `topic`: `是否把一个有争议的协作问题从 discussion 收口为可执行落点`
- `source_discussion`: [discussion.md](discussion.md)
- `why_active`: `当前 discussion 仍保留为边界线程，直到最终落点被正式接住`
