# discussion-decision-task

This is the official first-pass example pack for the discussion promotion path.

Canonical path only:

1. `active-discussions.md` as the entry page
2. `discussion.md`
3. `issue_ledger.md`
4. `decision_package.md`
5. `task_or_decision.md`

The pack also includes:

- a governed workflow contract
- a current workflow state and derived status projection
- object contracts for the discussion, issue ledger, decision package, and final landing

Use this pack as the stable shape for discussion closure and promotion, not as a project-specific policy.
